#### What is this?

这是一个小工具，可以快速生成ydl java project。

#### 用法


```

python gen.py  resource path/to/project

```


###### python about /work/user-center

此时将自动在user_center下建立如下文件:
```
- AboutReqDto.java
- AboutRespDto.java
- AboutFacade.java
- About.java
- AboutBiz.java
- AboutBizImpl.java
- AboutMapper.java
- AboutFacadeImpl.java
- AboutMapper.xml

```
